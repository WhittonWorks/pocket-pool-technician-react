# Pocket Pool Technician (React Build)

**React-based diagnostic app for pool service technicians ‚Äî JSON-driven, offline-ready, and built to scale across all pool equipment.**

---

## üìã Overview
Pocket Pool Technician (PPT) is a diagnostic app designed for **pool service technicians**.  
This version is built in **React**, providing a modern, scalable foundation for growth.  

The app uses **JSON-driven diagnostic flows** to guide technicians step by step through troubleshooting pool equipment including:
- Heaters
- Pumps
- Automation systems
- Filters
- Water care systems
- Lighting

---

## üöÄ Features
- **JSON-driven flows** ‚Üí Add new equipment by dropping in JSON files.  
- **Reusable React components** ‚Üí Clean, modular codebase.  
- **Offline support (PWA)** ‚Üí Works in backyards with no WiFi.  
- **Scalable structure** ‚Üí Easy to expand for all brands/models.  
- **Future-ready** ‚Üí Supports App Store/Google Play deployment with Capacitor.  

---

## üì¶ Tech Stack
- [React](https://react.dev/) (UI framework)  
- [TailwindCSS](https://tailwindcss.com/) (styling)  
- [Workbox](https://developer.chrome.com/docs/workbox) (service worker & offline caching)  
- [Capacitor](https://capacitorjs.com/) (for native app packaging ‚Äî later phase)
